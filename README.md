# commonR

This package contains helper functions and Seurat wrapper for scrna-seq analysis.

## Installation

```R
remotes::install_github(repo = "brickmanlab/commonR", tag = "0.1.0")
```

## From dev

```R
remotes::install_github(repo = "brickmanlab/commonR", ref = "dev")
```
